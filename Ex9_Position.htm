  3. Position Properties

    • Position: static - default: positioned base on normal flow of the page, not affected by top, bottom, left, right properties
    • Position: relative - adjust relative position from its normal(default) position by setting top, bottom, left, right properties
    • Position: fixed - Fixed element when scrolling, its position is set by top, bottom, left, right properties
    • Position: absolute - adjust relative position from its ancesstor position by setting top, bottom, left, right properties. If it have no ancestor, it will use the document body
    • Position: sticky - toggles between relative and fixed - relative until a given offset position is met in the viewport - then it sticks in place like fixed
    The <div> element has position: static
    The <div> element has position: relative
