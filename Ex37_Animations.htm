Animation-delay

animation-delay:-2s - Start as if it already been playing for 2s

Animation-iteration-count

animation-iteration-count

:3 - animation run 3 times

animation-iteration-count

:infinite - animation run infinitely

Animation-direction

  • normal - play as normal (backward) (default)
  • reverse - play in reverse direction (backward)
  • alternate - play forward first, then backward (set with animation-iteration-count to see)
  • alternate-reverse - play backward first, then forward (set with animation-iteration-count to see)

Animation-timing-function

  • ease - slow start, then fast, then end slowly
  • linear - same speed from start to end
  • ease-in - slow start
  • ease-out - slow end
  • ease-in-out - slow start, slow end
  • cubic-bezier(n,n,n,n) - dfine your own values in a cubic bezier function

    • Animation-fill-mode

    CSS animation not affect the element BEFORE FIRST FRAME and AFTER LAST FRAMES

    Can use animation-fill-mode to override it

    • none - default - BEFORE FIRST FRAME = AFTER LAST FRAME = Property set for that element (in class/id/style/...)
    • forwards - AFTER LAST FRAME = LAST FRAME
    • backwards - BEFORE FIRST FRAME = FIRST FRAME
    • both - combine forwards & backwards

    Detail and shorthand property

    Detail specify

      animation-name: example;
  animation-duration: 5s;
  animation-timing-function: linear;
  animation-delay: 2s;
  animation-iteration-count: infinite;
  animation-direction: alternate;

    Shorthand specify

      animation: example 5s linear 2s infinite alternate;
