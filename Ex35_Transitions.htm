Transistion-timing-function

  • ease - slow start, then fast, then end slowly (defaut)
  • linear - same speed from start to end
  • ease-in - slow start
  • ease-out - slow end
  • ease-in-out - slow start, slow end
  • cubic-bezer(n,n,n,n) - define your own value in a cubic bezier function

Transition-delay

transition-delay:1s: Delay 1sec before starting

Transition specified detail and shorthand

Detail specify

  transition-property: width;
  transition-duration: 2s;
  transition-timing-fuction: linear;
  transition-delay: 1s;

Shorthand specified

 transition: width 2s linear 1s;
