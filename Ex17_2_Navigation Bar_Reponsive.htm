Responsive Navigation Example

This example use media queries to transform the navigation to a top navigation bar when the screen size is 900px or less

We have also added a medi query for screens that are 400px or less, which will vertically stack and center the navigation links

You will learn more about medi queries and responsive web design later in our CSS Tutorial.

Resize the browser window to see the effect.
