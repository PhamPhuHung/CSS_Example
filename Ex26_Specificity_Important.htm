Specificity Hierachy

4 categories which define the specfificity level of a selector:

0. *

1. Elements, pseudo-elements - element name (h1), pseudo-element (::before, ::after)

10. Classes, attributes, pseudo-classes - .classes, [attributes], pseudo-class (:hover, :focus)

100. IDs - a unique identifier for page elements

1000. Inline styles - attached diretly to the element to be styled

Calculate Specficity

* -> 0;

h1, h1:last-child -> 1. If equal, the latest specification is used.

.ex1 -> 10, h1.ex1 -> 11

#ex1 -> 100, #ex1 h1 -> 101

<h1 style="color:red"%gt -> 1000
