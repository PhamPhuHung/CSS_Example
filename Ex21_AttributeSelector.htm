  5. The [attribute$="value"] selector

    6. Select elements whose attribute value ends with a specified value

    The following example selects all elements with a class attribute value that ends with "text"

    Note:The value does not have to be a whole word!

    Class="first_test"
    Class="second"

    Class="mytest"
