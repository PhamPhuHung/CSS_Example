1. Overflow: visible

    2. Visible - Default. The overflow is not clipped. THe content renders outside the element's box.

    You can use the overflow property when you want to have better control of the layout. THe overflow property specifies what happens if content overflows an element's box.




  2. Overflow: hidden

    3. Hidden - The overflow is clipped, and the rest of the content will be invisible.

  3. Overflow: scroll

    4. Scroll - The overflow is clipped and a scrollbar is added to see the rest of the content
